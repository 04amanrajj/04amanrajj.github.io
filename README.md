## 🛠️ Tech Stack & Architecture

This portfolio is engineered for optimal performance, accessibility, and modern aesthetics using:

*   **Core:** React 18, TypeScript, Vite
*   **Styling:** Tailwind CSS, shadcn/ui (Radix Primitives)
*   **Icons:** Lucide React, React Icons
*   **Analytics:** Microsoft Clarity (User behavior analysis & heatmaps)
*   **Deployment:** GitHub Pages (Automated CI/CD workflows)

## ⚙️ Local Development

### 1. Clone the repository
```bash
git clone https://github.com/04amanrajj/04amanrajj.github.io.git
cd 04amanrajj.github.io
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```

### 4. Build and Deploy
```bash
npm run deploy
```
